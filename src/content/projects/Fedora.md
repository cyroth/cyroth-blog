## Chapter one
I had some headaches with S.T.A.L.K.E.R 2 on Arch thanks to it not playing nice with some version of the Nvidia drivers so I thought I'd set up a "stable" distro.

Initially I went with Mint. Already being a long time Debian user, it seemed like a no brainer and things were fine for a while.

When you start digging around in it, you start to see the Mint on top of Ubuntu on top of Debian thing. Bits of each would just keep popping up and it just started to feel "off"

## Why Fedora?
With the community reputation it has, and the backing of Red Hat, I figured I'd try this next. Package availability has been great and its been rock-solid on the stability front.

Drivers and kernel are kept up to date pretty well.

Still running side-by-side with [Arch](/projects/move-to-arch), and kept a lot of the same workflows.
